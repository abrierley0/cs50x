SPELLCHECKER
============

_Written by Adam Brierley_
_Bedfordshire_
_UK_

_Updated: 3rd April 2026_

__This program will check the spelling of any text you upload.__

To use this code, reference a dictionary (a file containing a list of the correct spellings of all words) and the text of your choice as command line arguments.

The program will return the number of mispelled words, and some benchmarking statistics.
